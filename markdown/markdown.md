## 其他

### 扩展名

简而言之：推荐使用 .markdown。

Markdown 文件对扩展名没有强制要求，但大部分会采用下面这些扩展名：

.markdown|.mdown|.mkdn|.md|.mkd|.mdwn|.mdtxt|.mdtext|.text

推荐使用 .markdown 或 .mdown ，或 .md，不过 .md 在 linux 下也用作 machine descriptoin 文件。
