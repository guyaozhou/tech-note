### TortoiseGIT 如何用选择 ssh 客户端

* 右键： TortoiseGIT/Settings/Network/ssh Client
 * 如果要使用 OpenSSH，输入 ssh.exe 即可
 * 如果要使用 TortoiseGitPLink.exe，需要指定全路径
