#### 基本操作

- `parseInt('12')` -- 字符串 ==> 整数
- `parseFloat('3.14')` -- 字符串 ==> 浮点数
- `var x = 80; x.toString()` -- 整数 ==> 字符串
- `'>I6B5s'.replace(/(\d*.)/g, ' $1').split(' ').slice(1)` -- 分析 Python struct 格式字符串
  - 注意其中的： **`replace`** `$1` `split` `slice`
